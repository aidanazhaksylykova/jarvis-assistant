# PRD — JARVIS Personal Assistant
**Product Requirements Document**
Версия: 1.0 · Дата: 25 июня 2026 · Статус: MVP реализован

---

## 1. Обзор продукта

**JARVIS** — персональный браузерный ассистент для одного пользователя. Цель: помочь пользователю не забывать просьбы близких (жены, семьи) и личные задачи. Ассистент принимает команды голосом или текстом, находит оптимальные магазины, сравнивает цены и отправляет точные уведомления в заданное время.

**Ключевой сценарий:** Пользователь говорит «Джарвис, нужно купить карандаш для бровей» утром, указывает время напоминания 18:00 — и ровно в 18:00 получает уведомление с лучшим магазином и ценой.

---

## 2. Проблема

Пользователь регулярно забывает просьбы жены и личные поручения. Стандартные приложения-напоминалки:
- требуют установки
- неудобны для быстрого ввода
- не ищут магазины и не сравнивают цены
- не поддерживают голосовой ввод без настройки

---

## 3. Цели продукта

| # | Цель | Критерий успеха |
|---|------|-----------------|
| 1 | Быстрая фиксация задачи | Задача добавляется за ≤ 10 секунд голосом |
| 2 | Своевременное уведомление | Уведомление срабатывает точно в минуту |
| 3 | Работа без установки | Открывается как HTML-файл в браузере |
| 4 | Голосовое управление | Поддержка русского языка через Web Speech API |
| 5 | Сохранность данных | Задачи сохраняются при закрытии вкладки |

---

## 4. Пользователь

**Один пользователь** — взрослый мужчина, живёт в Астане, активно пользуется смартфоном. Технический уровень: базовый. Основные устройства: iPhone (Safari) и ПК (Chrome).

**Основные сценарии использования:**
- Жена попросила купить что-то → быстро говорит Джарвису → получает напоминание по дороге домой
- Нужно не забыть личное дело → добавляет задачу с точным временем

## 5. Функциональные требования

### 5.1 Голосовой ввод
- Активация нажатием на орб-кнопку (круглая иконка микрофона)
- Распознавание речи на русском языке (`lang: ru-RU`)
- Поддержка браузеров: Chrome, Safari (iOS/macOS), Edge
- Визуальная индикация: анимация волн + пульсирующая граница орба
- Автоматическое определение ключевых слов: «напомни», «купи», «нужно»
- Автоизвлечение времени из речи: «в 18:00», «в шесть вечера»

### 5.2 Текстовый ввод
- Поле ввода внизу главного экрана
- Отправка по Enter или кнопке «отправить»
- Та же логика обработки команд, что и у голосового ввода

### 5.3 Добавление задачи
- Поле текста задачи (свободный ввод)
- Выбор времени напоминания: 24-часовой формат, шаг 1 минута (00:00–23:59)
- Кнопка «Добавить и установить напоминание»
- Быстрые кнопки для частых задач (карандаш для бровей, список жены и др.)

### 5.4 Система уведомлений
- Запрос разрешения браузера (`Notification.requestPermission`) при первом запуске
- Системное push-уведомление через `new Notification()` — приходит даже при свёрнутой вкладке
- Голосовое объявление через Web Speech Synthesis при срабатывании
- Toast-уведомление внутри интерфейса
- Автоматический перенос на следующий день, если время уже прошло
- Ежедневное повторение до отметки «Выполнено»

### 5.5 Управление задачами
- Список всех задач с отображением: текст, время создания, время напоминания, статус
- Статусы: «Активно» (золотой бейдж) / «Выполнено» (зелёный бейдж)
- Действия над задачей: «Выполнено» и «Удалить»
- Выполненные задачи отображаются с пониженной прозрачностью

### 5.6 Список напоминаний
- Отдельная вкладка с активными (невыполненными) напоминаниями
- Отображение времени до срабатывания в реальном времени: «через 3 ч 20 мин»
- Обновление обратного отсчёта каждую минуту
- Удаление напоминания из списка

### 5.7 Голосовые ответы ассистента
- Все ответы Джарвиса озвучиваются через Web Speech Synthesis
- Автоматический выбор лучшего доступного голоса:
  - iOS/macOS Safari → Apple Milena (близко к Siri)
  - Android/Chrome → Google русский
  - Fallback → любой доступный ru-RU голос
- Параметры голоса: rate 0.92, pitch 0.85 (низкий, спокойный тембр)
- Кнопка «Тест голоса» для проверки

### 5.8 Интерфейс и навигация
- Три вкладки: Главная / Задачи / Напоминания
- Живые часы с секундами и датой на русском языке
- Бейдж текущего голоса с подписью платформы
- Баннер статуса разрешений уведомлений
- Toast-уведомления для системных сообщений (3.5 сек)

## 8. Архитектура реализации

```
jarvis.html
├── <style>          — вся стилизация (CSS переменные, компоненты)
├── HTML structure
│   ├── Шапка        — логотип, статус
│   ├── Часы         — живое время + дата
│   ├── Voice badge  — текущий голос + кнопка теста
│   ├── Perm banner  — статус разрешений уведомлений
│   ├── Tabs         — навигация между страницами
│   ├── Page: Home   — орб, bubble, форма задачи, быстрые кнопки, ввод
│   ├── Page: Tasks  — список задач с действиями
│   └── Page: Reminders — активные напоминания с обратным отсчётом
└── <script>
    ├── tickClock()        — живые часы (setInterval 1s)
    ├── pickVoice()        — умный выбор голоса
    ├── speak()            — голосовой синтез
    ├── requestNotifPerm() — запрос разрешений
    ├── fireNotification() — отправка уведомления
    ├── scheduleTask()     — точный таймер через setTimeout
    ├── restoreTimers()    — восстановление после перезагрузки
    ├── addTask()          — добавление задачи
    ├── renderTasks()      — рендер списка задач
    ├── renderNotifList()  — рендер напоминаний
    ├── startListen()      — Web Speech Recognition
    ├── processCmd()       — парсинг команд
    └── showToast()        — системные сообщения
```

## 9. Ограничения текущей версии (v1.0)

| Ограничение | Причина | Решение в v2.0 |
|-------------|---------|----------------|
| Вкладка должна быть открыта | Браузер не запускает JS в фоне | Service Worker + PWA |
| Поиск магазинов — демо-данные | Нет интеграции с API | Google Places API / 2GIS |
| Только один пользователь | Нет бэкенда | Node.js + база данных |
| Нет мобильных push-уведомлений | Ограничения браузера | React Native / PWA |
| Голос ≠ Siri / Джарвис из фильма | Авторские права | ElevenLabs API (свой ключ) |

## 10. Дорожная карта (будущие версии)

### v1.1 — PWA (Прогрессивное веб-приложение)
- Service Worker для фоновых уведомлений
- Установка на домашний экран телефона
- Работа офлайн

### v1.2 — Реальный поиск магазинов
- Интеграция с 2GIS API (Казахстан)
- Геолокация пользователя
- Сравнение цен в реальном времени
- Построение маршрута до магазина

### v2.0 — Полноценное приложение
- Бэкенд (Node.js / Supabase)
- Мультиустройственность — задачи синхронизируются
- ИИ-интеграция через Anthropic API
- Голос ElevenLabs (кастомный тембр)
- Список жены с совместным доступом

## 12. Глоссарий

**Орб** — круглая кнопка активации микрофона на главном экране.  
**Bubble** — блок с текстом последнего ответа Джарвиса.  
**Toast** — всплывающее уведомление внутри интерфейса (3.5 сек).  
**scheduleTask** — функция, устанавливающая точный таймер до времени напоминания.  
**fireNotification** — функция, отправляющая системный пуш + голосовое сообщение.
